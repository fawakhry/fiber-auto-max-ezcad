خطوات مهمة لحل مشكلة النسخة القديمة:
1) من GitHub افتح الريبو ثم Add file > Upload files.
2) ارفع كل الملفات أعلاه في جذر الريبو وليس داخل فولدر.
3) وافق على Replace للملفات القديمة.
4) بعد النشر افتح الرابط أول مرة بهذا الشكل:
https://fawakhry.github.io/fiber-auto-max-ezcad/?v=561
5) لو ظهر القديم اضغط Ctrl+F5 أو افتح من نافذة Incognito.
