# مطبعجي فايبر ليزر — V5.6.1

برنامج متصفح عربي لتجهيز صور الحفر على ماكينة فايبر 30W.

## مساران منفصلان

### 1) حفر صورة

- رفع صورة شخص كاملة.
- عزل AI احترافي باستخدام MODNet مع MediaPipe كخطة احتياطية.
- Alpha Matting للشعر والحواف وتقليل الهالة البيضاء.
- قص تلقائي قريب من الشخص بدون فريم أبيض زائد.
- تحكم جودة: تنظيف الخلفية، نعومة الحواف، تفاصيل الوجه، وقوة الأسود.
- توزيع إضاءة تلقائي لإزالة الظلال وتخفيف المناطق المحروقة قبل الحفر.
- زر توضيح مجاني باستخدام Swin2SR داخل المتصفح.
- تصدير صورة الشخص PNG على خلفية بيضاء ثابتة.
- تصدير حفر PNG / SVG / BMP / PDF بخلفية بيضاء.

### 2) حفر عين

- يفتح اختيار صورة جديدة مستقلة.
- اختيار «صورة كاملة» لاكتشاف الوجه وقص العينين والحاجبين تلقائيًا.
- اختيار «عين مقصوصة» لمعالجة القص الجاهز مباشرة.
- لا يعتمد على صورة الشخص في المسار الأول.
- توضيح AI اختياري ×2.
- وضع «محفظة واقعية» بتنقيط Stucki متدرج بدل الكتل السوداء.
- وضع Jarvis فوتوغرافي ووضع خطوط قوية.
- توزيع إضاءة الجلد والحفاظ على الجفن والرموش والقزحية والحواجب.
- اختفاء ناعم للجلد من الأطراف بدون فريم مستطيل.
- ضبط عرض الحفر الحقيقي وDPI قبل صناعة النقط.
- كثافة خفيفة افتراضيًا للجلد حتى لا تتحول العين لمستطيل أسود.
- كارت إعداد EZCAD يوضح إغلاق Dither وGray وLighten وPixel Power Adjust لأن الملف جاهز.
- قص الفراغات لأقصى حد.
- تصدير PNG على خلفية بيضاء أو SVG.

## النسخة التجريبية

افتح `demo.html`. المعالجة والمعاينة تعملان، لكن النتيجة تحمل علامة DEMO وأزرار التصدير مقفولة. النسخة الكاملة تفتح من `index.html`.

## تحديث النسخ المباعة

- زر «فحص التحديث» يقرأ `version.json` المنشور على GitHub Pages.
- عند إصدار نسخة جديدة غيّر رقم `version` والملاحظات داخل `version.json` وارفع الملفات إلى فرع `main`.
- النسخ الموجودة عند العملاء ستعرض التحديث ورابط تحميل أحدث ZIP تلقائيًا.
- لا تغيّر رابط `updateManifest` في `update-config.js` بعد البيع.

## التشغيل

يمكن فتح index.html مباشرة. للحصول على أفضل توافق مع موديلات AI يفضل تشغيل المجلد من localhost بالأمر:

    python -m http.server 8080

ثم افتح:

    http://localhost:8080/

أول استخدام يحتاج إنترنت لتحميل موديلات الذكاء الاصطناعي المجانية. بعد التحميل قد يستفيد المتصفح من الكاش. الصور تعالج داخل المتصفح ولا يتم رفعها إلى خادم البرنامج.

## GitHub Pages

ارفع كل الملفات، ومنها `index.html` و`demo.html` و`styles.css` و`app.js` و`update-config.js` و`version.json` و`.nojekyll` في جذر المستودع، ثم فعّل Pages من فرع main.
