# مطبعجي فايبر ليزر — V5

برنامج متصفح عربي لتجهيز صور الحفر على ماكينة فايبر 30W.

## مساران منفصلان

### 1) حفر صورة

- رفع صورة شخص كاملة.
- عزل AI احترافي باستخدام MODNet مع MediaPipe كخطة احتياطية.
- Alpha Matting للشعر والحواف وتقليل الهالة البيضاء.
- قص تلقائي قريب من الشخص.
- زر توضيح مجاني باستخدام Swin2SR داخل المتصفح.
- تصدير صورة الشخص PNG شفافة.
- تصدير حفر PNG / SVG / BMP / PDF.

### 2) حفر عين

- يفتح اختيار صورة جديدة مستقلة.
- المطلوب صورة مقصوصة للعين فقط.
- لا يعتمد على صورة الشخص في المسار الأول.
- توضيح AI اختياري ×2.
- استخراج الجفن والرموش والقزحية والحواجب.
- قص الفراغات لأقصى حد.
- تصدير PNG شفاف أو SVG.

## التشغيل

يمكن فتح index.html مباشرة. للحصول على أفضل توافق مع موديلات AI يفضل تشغيل المجلد من localhost بالأمر:

    python -m http.server 8080

ثم افتح:

    http://localhost:8080/

أول استخدام يحتاج إنترنت لتحميل موديلات الذكاء الاصطناعي المجانية. بعد التحميل قد يستفيد المتصفح من الكاش. الصور تعالج داخل المتصفح ولا يتم رفعها إلى خادم البرنامج.
